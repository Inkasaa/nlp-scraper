# NLP News Intelligence

This project scrapes news articles from the web and performs NLP analysis to extract insights. It is designed as a simple pipeline for collecting and enriching news data.

## Structure
- `scraper_news.py`: Scrapes news articles from online sources.
- `nlp_enriched_news.py`: Performs NLP analysis on the scraped news data.
- `data/`: Stores raw scraped news data.
- `results/`: Stores NLP-enriched results.
- `requirements.txt`: Lists required Python libraries.
